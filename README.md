## NucleStock - Sistema de Gerenciamento e Controle de Estoque

NucleStock é uma aplicação web voltada para o gerenciamento de estoque. 
O sistema permite cadastrar produtos, registrar movimentações de entrada e saída e acompanhar a disponibilidade dos itens de forma prática.
